# Question
Name three representatives of American modernism and explain key features.

# Thesis
Corpus notes frame American modernism through formal experimentation, fragmented narration, and intertextual technique.

# Core points
Modernism sections in the notes emphasize non-linear chronology, unstable meaning, multiple narrators, and allusions as defining techniques. This matters because form is presented as central to movement identity.

The same notes directly link T. S. Eliot and Ezra Pound to high-modernist literary practice, while broader sections include further period-relevant figures. This matters because representative naming should follow the source's own categorization.

A stronger answer distinguishes movement features from biography by tying each author to one formal characteristic. This prevents generic “important writer” descriptions.

# Mini-analysis
For oral exams, the best structure is: define modernism in one sentence, present three figures, then map each figure to one stylistic marker.
