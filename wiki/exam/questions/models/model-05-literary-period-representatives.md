# Question
Present at least three representatives and works from one literary period and explain key features.

# Thesis
A strong period answer combines representative authors, named works, and one formal feature that distinguishes the period.

# Core points
British lecture materials organize literary periods (for example Renaissance/Elizabethan-Jacobean and later developments) through author clusters and genre shifts. This matters because periodization is evidence-based in the corpus, not arbitrary.

American notes organize Romanticism, Realism, and Modernism through recurring names and movement labels. This matters because the same author can be used differently across movements depending on form and historical context.

For scoring, each representative should be linked to one work and one feature (e.g., narrative form, thematic focus, style). Without that third step, answers become memorized lists.

# Mini-analysis
Period questions reward comparative clarity: define the movement briefly, select three figures, then connect each figure to one concrete textual trait.
